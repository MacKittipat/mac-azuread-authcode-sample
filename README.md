## Description

Authorization Code sample with Azure AD

Azure AD Application name : AzSampleApp

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

http://localhost:3000/
